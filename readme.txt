== Description ==

[VidoopConnect][] makes it easier than ever to sign up and login to your site by
letting your visitors reuse their existing accounts from sites like Google,
Yahoo, AOL, Facebook and MySpace, or any other OpenID provider.

[VidoopConnect]: http://vidoopconnect.com/


== Installation ==

1. Install the [OpenID][] plugin for WordPress
1. Upload the `vidoopconnect` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. [Register your site][] with VidoopConnect, using the information from the 'VidoopConnect' Settings menu in WordPress

[OpenID]: http://wordpress.org/extend/plugins/openid/
[Register your site]: https://vidoopconnect.com/rp/
